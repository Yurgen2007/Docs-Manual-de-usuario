En la parte superior izquierda se encuentra el boton de "subir masivamente", que desplegara un modal con el cual el usuario podra descargar un archivo excel con los 
datos minimos para poder registrar varios usuarios a la vez, o simplemente cargar el archivo en caso de que ya lo tenga, tambien podemos ver dos botones,
"cancelar" en caso de que no quiera realizar la accion o "crear usuarios" cuando ya tenga el archivo excel seleccionado

<img src="/public/users/FormMasivo.jpg" alt="Descripción de la imagen" style={{ width: '500', maxWidth: '800px' }} />

Y al realizar esto automaticamente se le asignan unas contraseñas a esos usuarios,y en caso tal de que el usuario quiera cambiar su contraseña 
puede ver el apartado de <a href="/usuarios/editarperfil">editar datos del perfil</a>

De igual forma los usuarios registrados masivamente podran verse en la tabla de usuarios.
