<img src="/public/tipositio/crear.jpg" alt="Descripción de la imagen" style={{ width: '100%', maxWidth: '1200px' }} />

En la parte superior izquierda el usuario podra encontrar un boton azul "Añadir Tipo de sitio" el cual, nos ayuda a añadir un nuevo tipo de sitio al 
sistema, al darle clik este mostrará un formulario donde se le solicita llenar los datos basicos para el nuevo tipo de sitio como nombre. 


<img src="/public/tipositio/formcrear.jpg" alt="Descripción de la imagen" style={{ width: '50', maxWidth: '300px' }} />

Y al momento de darle en el boton de guardar, se podra ver el tipo de sitio registrado en la tabla.
